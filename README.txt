SKY-MULE submission package

How to use:
1. Open index.html in a browser to preview the website.
2. Upload the whole ZIP file to the submission portal.
3. If you need to add the proposal presentation, replace the placeholder with:
   - a Google Drive link, or
   - a compressed PDF version small enough to stay under the 20 MB limit.

Main files:
- index.html
- style.css
- files/ (included PDFs)

Note:
The original Proposal Presentation PPTX was not included because it is 47 MB and would break the submission size limit.
